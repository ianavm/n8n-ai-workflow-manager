1Executive Summary

The Team Allori AI Operations Platform automates the core operational tasks of a real estate team: handling client enquiries via WhatsApp and email, qualifying and routing leads to agents, scheduling property viewings, classifying and filing documents, and providing real-time performance dashboards via Telegram.

Key Outcome: Each agent saves approximately 40 hours/month of admin work. The system replaces the need for a dedicated admin assistant while providing 24/7 AI-powered client response capabilities. Total cost: R6,647/month for a 4-agent team (incl. VAT), compared to R32,000+ for equivalent manual admin support.

2System Architecture & Workflow Map

The system is organised into three layers: Intake channels (WhatsApp, Email, Telegram), Processing engines (AI, routing, scoring), and Background operations (scheduling, escalation, monitoring). All data flows through a central Google Sheets CRM with 14 tabs.

Processing Engine Sub-workflows

IDWorkflowWhat It Does
RE-02Lead RouterNormalises phone numbers, checks for existing leads (dedup), creates client + lead records, and passes to scoring engine
RE-03WhatsApp AI CommsGenerates context-aware AI replies using Claude Sonnet, strips markdown for plain text, sends via WhatsApp Cloud API, handles low-confidence handoff to human agent
RE-04Email AI CommsClassifies email intent with AI, generates professional email reply, logs thread for deduplication
RE-05Booking CoordinatorQueries agent's Google Calendar for free slots, validates 2-day minimum advance booking, creates calendar event, sends confirmation via WhatsApp
RE-06Document ClassifierAI classifies documents into 15 South African RE categories (FICA, OTP, Mandate, Title, Municipal, etc.) with confidence scoring
RE-08Document FilingDeduplicates by content hash, renames per convention (date_surname_type_ref.ext), uploads to Google Drive folder structure, creates CRM record
RE-10Team NotificationsSends formatted Telegram notifications to team members with email fallback if Telegram delivery fails
RE-15Scoring EngineScores leads 0-100 using weighted criteria: budget (20%), timeline (15%), pre-approval (15%), area (10%), requirements (10%), engagement (10%), responsiveness (10%), source (10%)
RE-16Assignment EngineScores available agents by area match (35%), specialisation (25%), workload (25%), round-robin (15%) and assigns the best-fit agent
RE-18Telegram Alert RouterFormats and routes system alerts to the appropriate Telegram chat based on severity and type
RE-19Document Pack SenderAssembles and sends buyer/seller/tenant/landlord document packs via email with Google Drive links
