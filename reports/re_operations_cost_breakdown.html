Usage Assumptions (4 Agents)

WhatsApp Messaging

  • Inbound enquiries: 15 per agent/day = 60/day total = ~1,800/month
  • AI auto-replies: ~70% of inbound = ~1,260/month FREE (service window)
  • Follow-up templates (utility): 3 per lead = ~450/month $0.0076 each
  • Appointment reminders (utility): 2 per booking = ~200/month $0.0076 each
  • Marketing templates: 50/month (listing alerts, open house invites) $0.0379 each
  • Stale lead re-engagement (marketing): ~100/month $0.0379 each

Email

  • Inbound emails: 20 per agent/day = 80/day total = ~2,400/month
  • AI-generated replies: ~60% = ~1,440/month (Gmail free with OAuth)
  • Document emails processed: ~200/month

AI (Claude Sonnet via OpenRouter)

  • WhatsApp AI replies: ~1,260 calls/month (avg 300 input + 500 output tokens)
  • Email AI replies: ~1,440 calls/month (avg 500 input + 500 output tokens)
  • Document classification: ~200 calls/month (avg 800 input + 1,000 output tokens)
  • Daily summaries + performance: ~35 calls/month (avg 1,000 input + 2,000 output tokens)
  • Lead scoring/qualification: ~1,800 calls/month (avg 400 input + 300 output tokens)

n8n Workflow Executions

  • WhatsApp intake + routing: 1,800/month (1 exec per message, triggers sub-workflows)
  • Email intake + routing: 2,400/month
  • Telegram commands: ~300/month (team usage)
  • Scheduled workflows: ~2,700/month (escalation every 15min = 2,880 + daily/weekly)
  • Total: ~7,200 executions/month
